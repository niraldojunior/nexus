# Button

## Finalidade

- Executar ações primárias, secundárias e destrutivas do sistema.

## Regras

- O botão primário usa o amarelo V.tal.
- O texto deve ser curto e imperativo.
- O hover deve ser discreto e responsivo.
- Estados desabilitado, carregando e foco devem ser explícitos.

## Orientação

- Usar botão primário para a ação principal da tela.
- Usar botão secundário para ações complementares.
- Evitar excesso de botões na mesma linha.
