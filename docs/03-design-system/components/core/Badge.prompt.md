Small uppercase label for statuses, categories, and counts.

```jsx
<Badge tone="green" dot>Online</Badge>
<Badge tone="brand">TM Forum</Badge>
```

Tones: `neutral`, `green`, `blue`, `amber`, `red`, `purple`, `brand` (yellow tint), `ink` (yellow-on-black). Set `dot` for a leading status dot. For network-element status prefer the richer `StatusPill`.
