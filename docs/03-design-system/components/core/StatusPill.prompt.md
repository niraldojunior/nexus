Status indicator tuned to the V.tal Nexus domain — element health and viability outcomes — with a colored dot and Portuguese label.

```jsx
<StatusPill status="online" pulse />
<StatusPill status="inviavel" />
<StatusPill status="parcial" label="Parcial · 3 portas" />
```

Status keys map to color: green (`online/viavel/ativo`), blue (`curso/sincronizando`), amber (`parcial/degradado`), red (`inviavel/offline`), purple (`planejado/reservado`). Set `pulse` for live elements; override copy with `label`.
