# Input

## Finalidade

- Capturar texto, códigos, números e filtros.

## Regras

- Labels devem ser claras e objetivas.
- O estado de foco precisa ser visível.
- Erros devem ser mostrados com mensagem direta.
- Inputs devem respeitar a densidade visual do produto.

## Orientação

- Usar máscara quando o domínio exigir formato fixo.
- Usar mono para códigos técnicos e IDs.
- Agrupar inputs relacionados com espaçamento consistente.
