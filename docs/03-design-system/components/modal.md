# Modal

## Finalidade

- Exibir decisões, confirmações e formulários contextuais.

## Regras

- O modal precisa bloquear a atenção do usuário sem exagero visual.
- A hierarquia de título, conteúdo e ação deve ser clara.
- Fechamento deve ser simples e previsível.
- A ação principal deve ficar destacada.

## Orientação

- Usar modal para confirmações importantes.
- Evitar modais longos quando uma página dedicada for melhor.
