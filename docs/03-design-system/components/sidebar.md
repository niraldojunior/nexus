# Sidebar

## Finalidade

- Ajudar na navegação principal e na leitura de contexto do produto.

## Regras

- A sidebar é escura e precisa sustentar o contraste da interface.
- O item ativo deve estar claramente destacado.
- A navegação deve ser curta e orientada a módulos.
- Ícones e rótulos devem trabalhar juntos.

## Orientação

- Usar sidebar como elemento estrutural da aplicação.
- Evitar submenus excessivamente profundos.
