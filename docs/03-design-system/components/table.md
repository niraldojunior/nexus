# Table

## Finalidade

- Exibir listas técnicas, inventário e resultados operacionais.

## Regras

- Cabeçalhos devem ser claros e curtos.
- Linhas precisam ser legíveis e densas.
- Ordenação e filtros devem ser previsíveis.
- Estados vazios e de carregamento precisam ser tratados.

## Orientação

- Usar tabelas para volumes médios e altos de dados.
- Usar chips e badges para estados.
- Usar cores de classe apenas onde houver significado técnico.
