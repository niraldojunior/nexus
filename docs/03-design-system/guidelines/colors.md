# Cores

## Principios

- Usar o amarelo V.tal como acento principal, com moderacao.
- Evitar preenchimentos extensos em amarelo.
- Priorizar superficies claras com contraste corporativo.
- Usar cores semanticas apenas para estados e feedback.

## Paleta base

- `--vt-yellow`: `#FFD919`
- `--vt-yellow-light`: `#FFE047`
- `--vt-ink`: `#181919`
- `--surface-app`: `#C7C9D2`
- `--surface-card`: `#FFFFFF`
- `--surface-sidebar`: `#2E3238`
- `--border`: `#E2E8F0`
- `--text-primary`: `#181919`
- `--text-secondary`: `#4B5563`

## Uso recomendado

- Fundo da aplicacao: cinza claro.
- Cards e paineis: branco.
- Navegacao lateral: escuro.
- CTA principal: amarelo V.tal.
- Texto sobre amarelo: sempre `ink`.
