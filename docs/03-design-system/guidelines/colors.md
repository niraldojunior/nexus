# Cores

## Princípios

- Usar o amarelo V.tal como acento principal, com moderação.
- Evitar preenchimentos extensos em amarelo.
- Priorizar superfícies claras com contraste corporativo.
- Usar cores semânticas apenas para estados e feedback.

## Paleta base

- `--vt-yellow`: `#FFD919`
- `--vt-yellow-light`: `#FFE047`
- `--vt-ink`: `#181919`
- `--surface-app`: `#E5E7EB`
- `--surface-card`: `#FFFFFF`
- `--surface-sidebar`: `#2E3238`
- `--border`: `#E2E8F0`
- `--text-primary`: `#181919`
- `--text-secondary`: `#4B5563`

## Uso recomendado

- Fundo da aplicação: cinza claro.
- Cards e painéis: branco.
- Navegação lateral: escuro.
- CTA principal: amarelo V.tal.
- Texto sobre amarelo: sempre `ink`.
