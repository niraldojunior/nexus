# Espaçamento

## Princípios

- Basear todo o sistema em uma malha de 4px.
- Manter espaçamento consistente entre componentes.
- Usar respiro maior entre blocos funcionais.
- Evitar valores arbitrários fora da escala definida.

## Escala

- `0`
- `4px`
- `8px`
- `12px`
- `16px`
- `24px`
- `32px`
- `48px`
- `64px`

## Uso

- Padding de componentes: 16px ou 24px.
- Gaps internos: 8px a 16px.
- Separação entre seções: 24px ou mais.
