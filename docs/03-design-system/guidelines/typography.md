# Tipografia

## Princípios

- Usar tipografia corporativa, legível e densa.
- Reservar fonte display para títulos e números principais.
- Reservar fonte mono para códigos, IDs e payloads.
- Evitar misturar muitas famílias tipográficas.

## Famílias

- UI: `Inter`
- Display: `Montserrat`
- Mono: `JetBrains Mono`

## Hierarquia

- Display para KPIs, cabeçalhos e navegação.
- Títulos de seção com forte peso visual.
- Corpo com leitura confortável e compacta.
- Rótulos, chips e cabeçalhos de tabela com tracking amplo.
