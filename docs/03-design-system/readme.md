# V.tal Nexus - Design System

Este diretório reúne as diretrizes visuais e de interface do V.tal Nexus.

## Objetivo

Padronizar o uso de cores, tipografia, espaçamento, princípios de interface e componentes para todo o produto Nexus.

## Fontes de referência

- Identidade visual do V.tal Nexus já existente neste diretório.
- Direcionadores técnicos do produto.
- Kits visuais e assets já produzidos para o Nexus.

## Diretrizes-base

- Priorizar consistência visual sobre criatividade local.
- Reutilizar componentes antes de criar novas variações.
- Manter linguagem técnica e corporativa.
- Evitar soluções genéricas que descaracterizem o produto.
- Manter a documentação em português do Brasil.
